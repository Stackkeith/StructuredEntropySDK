# StillnessEngine

This repository contains the source code and research materials for the StillnessEngine project.

*   `/paper`: LaTeX source and PDF of the research paper.
*   `/code`: Python and web simulations of the Stillness Engine.
*   `/docs`: Supporting documentation and diagrams.
*   `/LICENSE`: The project's open-source license.
*   `/.gitignore`: Files and directories to be ignored by version control.
